Parley

An object-oriented approach to concurrent programming

The Problem
  • Concurrent Hardware is becoming increasingly popular
  • But language support is inconsistent
    • Threads in Java and C++
    • Global Interpreter Lock in Ruby and Python
    • Actors in Scala and Io
    • CSP in Go
  • Some languages with built-in concurrency
    • Implicit parallelism in SISAL and Haskell
    • Actors in Erlang
The Proposal

A new language built for concurrent computation with familiar concepts

  • Purely object-oriented (based on Prototypes)
  • Implicitly parallel (objects as Actors)
  • Designed for uniformity and fliexibility
    • Homoiconic syntax ad object model
    • Dynamic typing and garbage collection
    • All language constructs are first class
The Actor Model
  • Established model of parallel computation
  • A program is a set of independent Actors
  • Actors respond to messages by
    • sending messages to other Actors
    • creating new Actors
    • designating behavior to be used for the next message
  • Nothing is shared. Only communication is via messages.
Prototypes
  • Objects without classes
  • Create new objects by cloning existing objects
    • New object is an exact copy of parent object
    • Behavior is shared by "delegation"
    • Removes inheritance and favors composition
  • Programs are a set of interacting objects
  • Popular languages include JavaScript, Lua and Io

Parley

... is an object-oriented, prototype based programming language using the Actor model for implicit parallelism and concurrency.
Roadmap
  • Create a syntax for object creation and actor messaging
  • Research and develop a model for mapping actors to threads and processes
  • Choose a target platform (probably an established concurrent VM)
  • Create a compiler for the target platform
  • Implement concurrent algorithms in Parley, tweaking language and compiler as needed
  • Implement equivalent algorithms on other languages and platforms
  • Test Parley implementation against other language and platform implementation
